recipz Tech Stack & Deployment

  • Frontend: React (Create React App) + Tailwind CSS + RadixUI/ShadCN + React Router
  • Backend: Flask + Firebase Admin SDK + Firestore + BeautifulSoup + lxml
  • Database: Firebase Firestore
  • Auth & Storage: Firebase Auth + Firebase Storage
  • Deployment: Render (backend) + Cloudflare Pages (frontend)
  • Scraping: Python (requests + BeautifulSoup + lxml)
  • API Style: RESTful (Flask)
  • PWA: CRA + manifest.json + service workers + offline caching
  • CI/CD: Manual deploy to Render + Cloudflare (no third-party CI specified)
  • SEO: Canonical slug-first URLs + dynamic metadata + 301 redirects
